2. Anatomy and grid theory

What is web page anatomy

Almost all web pages consist of the following parts: header, main content, stuff on the side, footer. We put header at the top of each page, and footer at the bottom. Stuff on the side may contain additional information, navigation, advertising, etc. The navigation system has to be consistent throughout the website. Main content is the biggest part of the page, and is unique to it. These elements are quite common in all websites but they can be laid out different ways which means responsive design.

What is grid theory

Many web pages are based on a grid system that means that the page's lyaout is based on columns. What is the width of each column? How to place elements according to the grid? Grid theory answers to these questions, and allows us to create proportioned design. Grid provides a framework for the site design as set of guidelines for how elements should be positioned within a layout. The elements may span two or more columns, the main idea is keeping them in proportion.

Grid system examples: page splits into 4 or 6 columns

Let's look at the examples of page layout from image 1, but organized in grid. For this purpose you need to do some calculations to determine the width of page's elements. For example, if the width of the page is 960 px, and you want section on the side to be one third, it's width is about 320 px. However, relative calculations are easier, you may use width in percentage for the elements. There are different useful tools that helps to calculate the grid, for example Grid Generator. You need to decide the number of columns you want in a row and set the margin you want to use, and the program will make all the calculations, and also will generate CSS and HTML code for this layout. You may use GridFox tool as Firefox broweser extension to create a grid.

Grid system and usability

Grid system increases usability, because it provides visual and structural balance of web pages. It is flexible, so you can easily change the element of the layout without redesign the whole page. Grid system supports responsive design, that means that your page looks good on any screen. This is easy and usefull tutorial how to create responsive grid-view design.

Grid system provides responsive design: desktop, tablet, phone

What if grid restricts my creativity?

If grid system restricts my creativity, I'll try to play around with the elements on a page adjusting them to the grid structure. I may divide my page into 12 sections and span two or more columns. In reality, grid provides variety of options for laying out elements on page, and I can find one that makes my ideas work.
