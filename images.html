5. Images

Images create content and add interest to our web page, primary purpose of an image is to attract user's attention. We can use images as elements in the foreground as well as background of a web page.

Image editing

If you take your own photos, the main things to understand are lighting, optimal exposure, image composition, making subject in focus. It is very important to become familiar with graphics software. The most popular of them is Photoshop, however there are open source image editors, the following article inrtoduces them: 5 Free Open Source Image Editors For Windows, Mac, and Linux. Web developer has to know how to crop and edit images before publishing it. You can compare the same photo before and after editing.

Editing image with croping, brightness, color

Legitimate Image Sources

There are Creative Commons (CC) licenses that provide rules for what you can and can't do with the image. This article introduces how to attribute creative commons licensed images by using good examples and explanation. Necessary steps include:

  • Credit the creator
  • Provide the title of the work
  • Provide the URL where the work is hosted
  • Indicate the type of license it is available under and provide a link to the license

We can use images from Wikipedia Commons - databases of millions freely usable media files to which anyone can contribute. There are examples of the images from Wikipedia Commons with attribution.

The examples of images' attribution

There are many other sources of images that you can use freely, for example openclipart. It's useful to know that Google offers an Advanced Search option that helps you filter their google images to only those that are Free License, royalty free, or open to the public for use. Royalty free images mean that once a license fee is paid, the images may be used many times without paying additional fees.

However using clip art images when designing a web site for a client is not a good idea because content and images of the professional site must be authentic and real. Stock photography makes web site look cheap and easy.

Image file format
