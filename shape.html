4. Lines, Shapes, and Shadows

Elements of art

Lines, shapes, and shadows create site's atmosphere. Lines help to structure a site, to underline important parts of layout. Shapes attracts user's attention first of all. Shadows helps to create depth of a site and makes it more realistic.

Lines

Lines are used to underline some piece of text, to attract user's attention to some point. The most common usage of lines are links, we can use :hover selector to underline a link when you mouse over it. We also use lines to divide content of page into blocks that help organize layout. I've used Wordpress themes to show examples, so you can see how lines makes layout well-organized.

Using of lines to organize layout: Twenty Sixteen and Magazine lite themes

Shapes

Shapes and forms let you create original design with circles, stars, hearts, etc. Here are some of the ways in which you can make a shape on the web: border radius, rotating shapes, wrapping text into shapes. I think shapes and colors are those elements that make your web page original, unique.

Original design using shapes: Desk Mess Mirrored and Vantage themes

In the following examples I float the image left, give it a width, height and a margin then use the property shape-outside to curve the text around the circle and ellipse. I've used this example. However this property is still very new and now it works only in Chrome.

CSS shapes using shape-outside property

I've used CSS Clip Path Maker to make complex shapes with my images. I recommend this tool to create different shapes, you can animate them as well.

Examples of clip path maker: Octagon and Left Chevron Shapes

Rectangle shapes with text and image are used for layout, it is so called card design. With the rise of Pinterest, designers have been amazed by cards that help distribute information in a visual way so the visitors can easily read content.

Examples of card design: Almond lemon cookies and Pingraphy themes

Shadows

Light and shadow can add qualities like depth, realism, dimension to your design. It was popular in web design till 2013, when Apple shifted to flat design. Flat means the absence of shadows, strokes and gradients. It means simplicity in design, it also makes pages to be loaded more quickly, that is very important for mobile users. You can see that all of the Wordpress themes are flat, but it doesn't mean lack depth, dimension and realism. There are different techniques to add depth in flat design: color, typography, images.

Flat design with "depth" feeling Spacious and Twenty Eleven themes
